# Categorías de clientes

Categorías iniciales para clasificar los comercios clientes de Envax.

## Restaurante

| Subtipo | Descripción |
|---|---|
| Parrilla | Restaurante especializado en carnes a la parrilla |
| Chifa | Restaurante de comida china-peruana |
| Cevichería | Especializado en ceviche y pescados |
| Pollería | Restaurante especializado en pollo a la brasa |
| Comida criolla | Restaurante de comida peruana tradicional |
| Comida marina | Restaurante de pescados y mariscos |
| Comida italiana | Restaurante italiano |
| Comida internacional | Cocina de otros países |
| Café / Cafetería | Especializado en café y repostería ligera |

## Pollería

| Subtipo | Descripción |
|---|---|
| Pollo a la brasa | Venta de pollo a la brasa |
| Pollo broaster | Venta de pollo frito / broaster |
| Pollería con restaurante | Mixto: venta y consumo en local |

## Comida rápida

| Subtipo | Descripción |
|---|---|
| Hamburguesas | Hamburguesería |
| Salchipapas | Puesto de salchipapas |
| Sandwich | Sandwichería |
| Pizza | Pizzería |
| Anticuchos | Puesto de anticuchos |

## Carnicería

| Subtipo | Descripción |
|---|---|
| Carnicería de res | Venta de carne de vacuno |
| Carnicería de cerdo | Venta de carne de cerdo |
| Carnicería mixta | Venta de carnes variadas |

## Charcutería

| Subtipo | Descripción |
|---|---|
| Embutidos | Venta de embutidos |
| Jamones | Venta de jamones y fiambres |
| Mixta | Charcutería variada |

## Panadería

| Subtipo | Descripción |
|---|---|
| Panadería artesanal | Panes elaborados artesanalmente |
| Panadería industrial | Producción en volumen |
| Panadería con pastelería | Mixta con repostería |

## Repostería

| Subtipo | Descripción |
|---|---|
| Pastelería | Tortas y pasteles |
| Postres por porción | Postres individuales |
| Heladería | Helados y similares |

## Frutería

| Subtipo | Descripción |
|---|---|
| Frutería tradicional | Venta de frutas y verduras |
| Orgánica | Productos orgánicos |
| Mixta | Frutas + abarrotes |

## Ferretería

| Subtipo | Descripción |
|---|---|
| Ferretería general | Herramientas y materiales de construcción |
| Ferretería industrial | Insumos para la industria |
| Acabados | Acabados para construcción |

## Zapatería

| Subtipo | Descripción |
|---|---|
| Zapatería de calzado | Venta de zapatos |
| Reparación de calzado | Servicio de arreglo |
| Tienda de accesorios | Accesorios de cuero y similares |

## Tienda

| Subtipo | Descripción |
|---|---|
| Tienda de abarrotes | Venta de productos de primera necesidad |
| Bodega | Tienda de barrio |
| Tienda especializada | Ropa, electrónicos, etc. |
| Tienda por departamentos | Varias categorías |

## Micromercado

| Subtipo | Descripción |
|---|---|
| Micromercado de barrio | Autoservicio pequeño |
| Minimarket | Similar a micromercado |
| Autoservicio | Con o sin atención al público |

## Institucional

| Subtipo | Descripción |
|---|---|
| Colegio | Institución educativa |
| Universidad | Educación superior |
| Hospital / Clínica | Centro de salud |
| Iglesia | Institución religiosa |
| Municipalidad | Gobierno local |
| ONG | Organización no gubernamental |
| Fuerzas armadas / Policía | Instituciones castrenses |

## Servicios

| Subtipo | Descripción |
|---|---|
| Servicio técnico | Reparación de equipos |
| Taller mecánico | Mantenimiento de vehículos |
| Peluquería / Barbería | Servicios de cuidado personal |
| Lavandería | Servicio de lavado |
| Fotografía | Estudio fotográfico |

## Distribuidor

| Subtipo | Descripción |
|---|---|
| Distribuidor de alimentos | Distribuye productos alimenticios |
| Distribuidor de bebidas | Distribuye bebidas en general |
| Distribuidor mayorista | Venta al por mayor |
| Distribuidor logístico | Servicios de distribución |

## Otro

| Subtipo | Descripción |
|---|---|
| Sin subtipo definido | Comercio no clasificable en las categorías anteriores |

## Pendiente

| Subtipo | Descripción |
|---|---|
| Sin clasificar | Cliente pendiente de clasificación manual |
| En revisión | Cliente siendo evaluado para asignar categoría |
